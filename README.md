# PROYECTO 4:Triple Peaks coffee shop

## Descripción

Triple peaks coffee shop es un página de una libreria para estudiar, puedes encontrar recetas de preparacion de café y una seccion de reservaciones para poder ir a estudiar presencialmente a la biblioteca Triple Peaks, esta creada con html y css.
